zap O vision

Ce projet est le site web de mon portfolio. On y regarde mes projets au travers de la zapOvision, une télé avec laquelle on peut intéragir via ses boutons et le scroll de la souris. Ce projet m'a permis de développer mes compétences en html et css ainsi que d'apprendre à utiliser le javascript.

Points importants: HTML, CSS, JavaScript

It's a Sharky Day

Ce jeu est mon premier gros projet en C#. Cette expérience m’a permis de prendre en main Unity et d’utiliser les fonctionnalités du moteur ainsi que de créer mes propres assets 2D avec Gimp.

Points importants: Unity, C#, UI, Animator, Gimp

Just Waves

J’ai programmé ce jeu en python en utilisant PyGame. Ce projet m’a permis d’apprendre la programmation orientée objet. Pour ce projet, j’avais comme but de tout faire moi même sauf les sons donc j’ai appris beaucoup de choses en faisant mon propre système de UI, de collision, de rendu de texte en dessinant ma propre police.

Points importants: Python, PyGame

IA quoi

Ce jeu a été réalisé pour l'édition 2025 de la scientific game jam à Toulouse. Cette jam a pour but de gamifier la thèse d'un doctorant pour ensuite présenter le jeu au public de la cité de l'espace.

Points importants: Unity, C#

Blown Away

Ce jeu a été réalisé pour la Global Game Jam 2025 à Toulouse en 2 jours.

Points importants: Unity, C#

Kill The Francis

J’ai fait ce jeu pendant un week end en équipe de 3 pour la microjam 29. J’ai appris à utiliser l’object pooling pour optimiser les performances de la foule.

Points importants: Unity, C#

Bubble Gnome

J’ai produit ce jeu durant la gamejam de l’Enjmin (Enjam) en équipe avec des étudiants de masters en 3 jours.

Points importants: Unity, C#

Application Flask

C’est un projet de NSI dans lequel on a créé une base de données avec laquelle on peut interagir via une interface web. Pour ce projet, notre équipe a décidé d’utiliser Flask afin de répertorier les bonus, ennemis et armes de chaque niveau de la première campagne de Doom 1993.

Points importants: python, flask, html, css, sql, bases de données

4H Runner

C'est un jeu que j'ai fait pour m'entrainer à prototyper rapidement un projet. Ce runner a été réalisé en 4h.

Points importants: Unity, C#

Unity SRP

Ce projet m'a permis de mieux comprendre le fonctionnement des Scriptable Render Pipeline de Unity en créant le mien. J'ai pu mettre en place mon render pipeline ainsi qu'écrire des shader en HLSL réagissant aux directional lights. Ce projet n'est pas encore terminé, je suis en train de travailler sur les ombres.

Points importants: Unity, C#, HLSL

La zapOvision est une télévision révolutionnaire! Située dans votre navigateur, elle vous permettra de voir tous mes projets!

Utilisez votre souris comme une télécommande! Scrollez pour parcourir les 3 chaines de la zapOvison et pointez sur un projet pour l'agrandir!

Cette merveille comprend aussi des boutons qui vous permettront d'aller voir les pages itch et le code de mes projets mais aussi d'éteindre la télé si vous en avez eu assez!
